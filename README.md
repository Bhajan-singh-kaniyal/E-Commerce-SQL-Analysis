# 🛒 E-commerce Data Analysis using SQL

## 📌 Overview

This project analyzes an e-commerce dataset to understand customer behavior, revenue trends, and product performance using SQL.

---

## 🎯 Objective

* Identify high-value customers
* Analyze revenue trends
* Detect customer churn
* Improve business decision-making

---

## 📂 Dataset

* Olist E-commerce Dataset (Kaggle)
* 100K+ records including customers, orders, payments, and products

---

## 🛠️ Tools

* SQL (PostgreSQL / MySQL)
* CSV Files

---

## 🔍 Key Analysis

* Revenue trends (monthly sales)
* Customer segmentation (RFM analysis)
* Churn analysis (inactive customers)
* Top-selling products

---

## 💡 Key Insights

* A small group of customers generates most revenue
* Sales increase during festive periods
* Some customers become inactive after a few months
* Few products contribute to most sales
