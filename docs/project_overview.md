# 📘 Project Overview

## 🎯 Objective

Analyze e-commerce data to understand customer behavior and improve business decisions.

---

## 🛠️ Approach

* Data cleaning using SQL
* Data analysis (joins, aggregations, window functions)
* Customer segmentation (RFM)
* Churn detection

---

## 📊 Key Outcomes

* Identified high-value customers
* Found revenue trends
* Detected inactive (churn) customers
* Analyzed top-performing products

---

## 🚀 Tools Used

* SQL (PostgreSQL/MySQL)
* CSV Dataset
* GitHub
